The `enchanted-popover` displays contextual content in a floating panel anchored to a target element. Supports 13 arrow positions, dark/light themes, optional close buttons, and full RTL support for flexible positioning.

## Slots
| **Slot** | **Description** |
|----------|-----------------|
| `target` | **Required.** Used to provide the element that triggers the popover on hover or keyboard focus. If omitted, the popover has no target and cannot be displayed. |
| `label` | **Optional.** Used to provide custom label content when `showLabel` is enabled. If omitted, the component displays the value of the `label` property. |
| `text` | **Optional.** Used to provide custom popover body content when `showText` is enabled. If omitted, the component displays the value of the `text` property.|
