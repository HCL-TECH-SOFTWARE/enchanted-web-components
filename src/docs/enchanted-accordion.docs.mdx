## Slots
| **Slot** | **Description** |
|----------|-----------------|
| `header` | **Optional.** Used to provide custom content for the accordion header. If omitted, the component displays its default header text. | 
| `secondary` | **Optional.** Used to provide custom secondary text displayed beneath or alongside the main header. If omitted, the component displays the value of the `secondaryText` property or its default localized secondary text when enabled. |
| `accordion-items` | **Required.** Used to provide the accordion content that is displayed when the accordion is expanded. If omitted, the expanded accordion has no content to display. |
