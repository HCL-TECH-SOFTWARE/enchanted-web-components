The `enchanted-toggle-button-group` is a container for multiple toggle buttons in horizontal or vertical layout. Supports single selection mode with optional badges, tooltips, and size customization for toolbar-style button groups.

## Slots
| **Slot** | **Description** |
|----------|-----------------|
| `default` | **Required.** Used to provide one or more `enchanted-toggle-button` components that participate in toggle group. If omitted, the toggle group is rendered without any buttons and cannot be interacted with. |
