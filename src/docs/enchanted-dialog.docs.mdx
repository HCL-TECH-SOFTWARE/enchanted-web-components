## Slots
| **Slot** | **Description** |
|----------|-----------------|
| `title` | **Optional.** Used to provide custom dialog header content when `overrideTitle` is enabled. If omitted, the component displays the value of the `dialogTitle` property. |
| `content` | **Required.** Used to provide the primary content displayed inside the dialog. If omitted, the dialog is rendered without any body content. |
| `pagination` | **Optional.** Used to provide pagination controls below the dialog content. If omitted, the pagination section remains empty.|
| `footer` | **Optional.** Used to provide action buttons or other footer content. If omitted, the dialog is rendered without footer actions. |
